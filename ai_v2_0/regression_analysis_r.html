Simple Linear Regression Analysis (R)¶

This notebook analyzes the relationship between years of experience and salary using a simple linear regression model.

Step 1: Load the Data¶

We read the CSV file into an R data frame. Each row represents an observation with predictor and response variables.

Step 2: Create a Scatter Plot¶

A scatter plot helps us visualize the relationship between years of experience (predictor) and salary (response).

In [2]:
x_column <- "YearsExperience"
y_column <- "Salary"

plot(
  data[[x_column]],
  data[[y_column]],
  xlab = x_column,
  ylab = y_column,
  main = paste(y_column, "vs", x_column),
  pch = 19,
  col = adjustcolor("steelblue", alpha.f = 0.7)
)

Step 3: Fit a Simple Linear Regression Model¶

We fit the model Salary = beta_0 + beta_1 * YearsExperience using ordinary least squares with lm().

In [3]:
formula <- as.formula(paste(y_column, "~", x_column))
model <- lm(formula, data = data)
summary(model)

Step 4: Overlay the Regression Line¶

We plot the fitted regression line on top of the scatter plot to visualize how well the model captures the trend.

In [4]:
plot(
  data[[x_column]],
  data[[y_column]],
  xlab = x_column,
  ylab = y_column,
  main = paste("Linear Regression:", y_column, "vs", x_column),
  pch = 19,
  col = adjustcolor("steelblue", alpha.f = 0.7)
)
abline(model, col = "red", lwd = 2)
legend("topleft", legend = c("Observations", "Regression line"), col = c("steelblue", "red"), pch = c(19, NA), lty = c(NA, 1), lwd = c(NA, 2), bty = "n")

Step 5: Evaluate the Model¶

We assess model performance using R-squared (proportion of variance explained) and RMSE (typical prediction error).

In [5]:
r2 <- summary(model)$r.squared
rmse <- sqrt(mean(residuals(model)^2))

cat(sprintf("R-squared: %.4f\n", r2))
cat(sprintf("RMSE: %.4f\n", rmse))

Summary¶

The linear model estimates how salary changes with years of experience. R-squared and RMSE summarize overall fit and prediction accuracy.
